## Cambios aplicados en este pase (C, D, E)

### C — Mensaje default de pauta → `saludo` (código)
Archivo: `src/lib/agents/varixcenter/comprehension-prompt.ts`.
- El anuncio pasará a pre-llenar "Hola! Me interesa una valoración". Ese texto es automático, no intención real.
- Regla nueva (modelada en el manejo de saludos de somnio-sales-v4): cuando el cliente ABRE la conversación (sin datos previos, sin pregunta del bot) con el texto default o variante de apertura genérica → `intent = saludo` (no `quiero_agendar`, no `precio_valoracion`). El bot responde bienvenida + "¿Deseas agendar tu valoración?" y deja que confirme; el "sí" posterior dispara `quiero_agendar` (regla post-saludo D-12). Si la apertura trae una pregunta concreta (precio/ubicación/síntoma), esa es primary y saludo es secondary.
- Añadido Ejemplo 4 al prompt.
