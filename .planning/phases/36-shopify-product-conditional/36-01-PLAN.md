Add a new helper function `resolveConditionalProducts` in the CRM Order Actions section of action-executor.ts (above `executeCreateOrder` or right below it as a private helper). The function:

```typescript
async function resolveConditionalProducts(
  config: {
    source: string  // Already resolved to actual value by resolveVariablesInObject
    mappings: Array<{ when: string; productId: string; quantity?: number }>
    defaultProductId?: string | null
    defaultQuantity?: number
  },
  workspaceId: string
): Promise<Array<{ productId: string; sku: string; title: string; unitPrice: number; quantity: number }>>
```

Implementation details:
1. **Numeric normalization for matching**: Try `parseFloat(source)` and `parseFloat(when)`. If BOTH are valid numbers (not NaN), compare as `sourceNum === whenNum`. Otherwise fall back to `String(source) === String(when)`. This handles the Shopify "109994.80" vs user "109994.8" pitfall.
2. Use `config.mappings.find()` with the above comparison.
3. Determine productId: `match?.productId ?? config.defaultProductId`. If null/undefined, return empty array.
4. Determine quantity: `match?.quantity ?? config.defaultQuantity ?? 1`.
5. Fetch product from DB: `createAdminClient()` -> `.from('products').select('id, sku, title, price').eq('id', productId).eq('workspace_id', workspaceId).eq('is_active', true).single()`.
6. If product not found, log warning and return empty array (graceful degradation, do NOT throw).
7. Return array with single product: `[{ productId: product.id, sku: product.sku, title: product.title, unitPrice: product.price, quantity }]`.

Then update `executeCreateOrder` to support 3 mutually exclusive product modes. Replace the current `const products = params.copyProducts && ...` block (lines 392-399) with:

```typescript
// Product resolution: 3 mutually exclusive modes
let products: Array<{ productId?: string | null; sku: string; title: string; unitPrice: number; quantity: number }> | undefined

if (params.productMappings && typeof params.productMappings === 'object') {
  // Mode 3: Conditional product mapping (takes precedence)
  products = await resolveConditionalProducts(
    params.productMappings as {
      source: string
      mappings: Array<{ when: string; productId: string; quantity?: number }>
      defaultProductId?: string | null
      defaultQuantity?: number
    },
    workspaceId
  )
  // Empty array means no match — treat as undefined for domain layer
  if (products.length === 0) products = undefined
} else if (params.copyProducts && Array.isArray(context.products)) {
  // Mode 2: Copy from trigger (existing behavior)
  products = (context.products as Array<{ sku: string; title: string; quantity: number; price: string; discounted_price?: string }>).map(p => ({
    sku: p.sku || '',
    title: p.title,
    unitPrice: parseFloat(p.discounted_price || p.price) || 0,
    quantity: p.quantity,
  }))
}
// Mode 1: No products (products remains undefined)
```
