## Technical Details

**PostponementBadge:**
- Returns null for count=0 (clean UI)
- Yellow badge for 1-2 postponements (warning)
- Red badge for 3+ postponements (critical)
- Tooltip explains the count in Spanish

**TaskNotesSection:**
- Full CRUD with optimistic updates
- handleCreate/handleSaveEdit/handleDelete all revert on error
- canModify permission check (author OR admin/owner)
- Empty state with icon and message

**TaskHistoryTimeline:**
- Icon mapping for all action types
- Spanish labels for all actions
- ActivityChanges helper formats field-level diffs
- Shows note preview for note-related actions
