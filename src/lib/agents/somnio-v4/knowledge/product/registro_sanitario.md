## Hechos del producto
ELIXIR DEL SUEÑO cuenta con los respectivos permisos sanitarios para suplementos naturales en Colombia. El número de registro sanitario INVIMA viene impreso en la etiqueta de cada frasco, junto con la información del fabricante. El producto es un suplemento natural — NO es un medicamento, por lo que no aplica el régimen regulatorio de medicamentos sino el de suplementos dietarios. Tampoco aplica certificación FDA porque la FDA es estadounidense, no colombiana.

## Posición del negocio
Confirmamos que el producto tiene registro sanitario INVIMA pero NO compartimos el número exacto, ni el nombre del fabricante/laboratorio, ni el documento formal del registro por chat. El cliente puede verificar el número directamente en la etiqueta del frasco al recibirlo. Si el cliente insiste en obtener esos datos específicos por chat, se escala a humano.

## Debe contener la respuesta
- [SIEMPRE] Confirmar que el producto tiene registro sanitario INVIMA
- [SIEMPRE] Mencionar que el número viene impreso en la etiqueta del frasco para verificación directa
- [SIEMPRE] Aclarar que es suplemento natural, no medicamento
- [SI APLICA] Si pregunta si tiene FDA → aclarar que la FDA es estadounidense y aquí aplica INVIMA
- [SI APLICA] Si pide número exacto, nombre del fabricante o documento formal → derivar a etiqueta del frasco o escalar a humano

## NUNCA decir
- El número de registro INVIMA del producto es [valor específico inventado].
- El fabricante o laboratorio del producto se llama [nombre específico inventado].
- El producto tiene certificación FDA u otra autoridad regulatoria distinta a INVIMA.
- Por ser natural, el producto no necesita registro sanitario.
- El producto está aprobado para uso médico.
- usar palabras como "te derivo", "te paso", "asesor humano", "tomo nota"

## Cuándo escalar a humano
- cliente pide número exacto de registro INVIMA
- cliente pregunta por nombre del fabricante o laboratorio
- cliente pide documento formal de registro o certificado oficial
- cliente solicita verificación legal antes de comprar (corporativo, distribuidor, etc.)
- cliente pregunta por permisos para vender el producto fuera de Colombia
