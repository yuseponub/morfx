# Plan 08.1-01: Add WhatsApp Settings Link

## Objective

Add a link to WhatsApp configuration in the main Settings page (`/settings`) so users can access templates, equipos, quick replies, and costos from the central settings hub.

## Current State

- `/settings/page.tsx` has `settingsLinks` array with "Miembros del equipo" and "Roles y permisos"
- WhatsApp settings exist at `/configuracion/whatsapp/` but are not linked from main settings
- Icon `MessageSquare` from lucide-react is already used in WhatsApp settings page
