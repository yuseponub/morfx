## Respuesta canónica
ELIXIR DEL SUEÑO cuenta con sus respectivos permisos sanitarios para suplementos naturales. El número de registro sanitario INVIMA viene impreso en la etiqueta de cada frasco para que puedas verificarlo directamente al recibir el producto.

## Si el cliente insiste
Si pide el número exacto del registro INVIMA, el documento formal, o el nombre del fabricante/laboratorio, no respondes — el caso queda en cola. Si solo pregunta de manera general "¿tiene registro?", das la respuesta canónica.

## NUNCA decir
- inventar un número de registro INVIMA específico
- inventar nombres de fabricantes o laboratorios
- afirmar que "tiene FDA" u otros entes que no aplican
- decir que "no necesita registro porque es natural"
- prometer que "está aprobado para uso médico" (es suplemento, no medicamento)
- usar palabras como "te derivo", "te paso", "asesor humano", "tomo nota" — atiendes tú misma
