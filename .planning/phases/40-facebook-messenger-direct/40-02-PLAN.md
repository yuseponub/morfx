`sendMessengerText(accessToken, pageId, psid, text, tag?: 'HUMAN_AGENT')`:
```typescript
const body = tag
  ? { messaging_type: 'MESSAGE_TAG', tag: 'HUMAN_AGENT', recipient: { id: psid }, message: { text } }
  : { messaging_type: 'RESPONSE', recipient: { id: psid }, message: { text } }
return metaRequest<{ message_id: string; recipient_id: string }>(
  accessToken, `/${pageId}/messages`, { method: 'POST', body: JSON.stringify(body) }
)
```
`psid` is a STRING — pass it verbatim into `recipient.id`. NEVER `Number()` it (PSID can exceed Number.MAX_SAFE_INTEGER, Pitfall 5).

`sendMessengerImage(accessToken, pageId, psid, imageUrl, tag?: 'HUMAN_AGENT')`:
```typescript
const body = {
  messaging_type: tag ? 'MESSAGE_TAG' : 'RESPONSE',
  ...(tag ? { tag: 'HUMAN_AGENT' } : {}),
  recipient: { id: psid },
  message: { attachment: { type: 'image', payload: { url: imageUrl, is_reusable: true } } },
}
// NO caption field — the image payload has none. Caller sends caption as a follow-up text.
```
