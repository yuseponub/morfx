Create `src/lib/agents/media/types.ts` with these types:

```typescript
export interface MediaGateInput {
  messageType: string       // 'text' | 'audio' | 'image' | 'video' | 'sticker' | 'reaction'
  messageContent: string    // text body, emoji for reactions, '[Audio]' for media
  mediaUrl: string | null   // Supabase Storage public URL (already re-hosted by webhook handler)
  mediaMimeType: string | null
  workspaceId: string
  conversationId: string
  phone: string
}

export type MediaGateResult =
  | { action: 'passthrough'; text: string }          // Continue to normal pipeline
  | { action: 'handoff'; reason: string }            // Handoff to human agent
  | { action: 'notify_host'; reason: string }        // Notify host only, bot stays active
  | { action: 'ignore' }                              // Silently ignore
```

Create `src/lib/agents/media/reaction-mapper.ts` with:

1. A `ReactionAction` type: `{ type: 'text'; text: string } | { type: 'notify_host'; reason: string } | { type: 'ignore' }`

2. An exported `REACTION_MAP` constant (Record<string, ReactionAction>) with the extended mapping from CONTEXT.md:
   - thumbs up (U+1F44D) -> { type: 'text', text: 'ok' }
   - red heart (U+2764 U+FE0F) -> { type: 'text', text: 'ok' }
   - laughing (U+1F602) -> { type: 'text', text: 'jaja' }
   - folded hands (U+1F64F) -> { type: 'text', text: 'gracias' }
   - crying (U+1F622) -> { type: 'notify_host', reason: 'Reaccion triste del cliente' }
   - angry (U+1F621) -> { type: 'notify_host', reason: 'Reaccion de enojo del cliente' }
   Also add heart variant without U+FE0F (some clients send plain U+2764).

3. An exported `mapReaction(emoji: string): ReactionAction` function that looks up the emoji in REACTION_MAP and returns `{ type: 'ignore' }` for unmapped emoji.

4. An exported `reactionToMediaGateResult(action: ReactionAction): MediaGateResult` function that converts ReactionAction to MediaGateResult:
   - text -> { action: 'passthrough', text: action.text }
   - notify_host -> { action: 'notify_host', reason: action.reason }
   - ignore -> { action: 'ignore' }
  </action>
