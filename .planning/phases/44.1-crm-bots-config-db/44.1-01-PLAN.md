## getPlatformConfig — NEW (Task 2)
```typescript
export const PLATFORM_CONFIG_TTL_MS = 30_000

export async function getPlatformConfig<T>(
  key: string,
  fallback: T,
): Promise<T>

export function invalidatePlatformConfigCache(key?: string): void
```
Contract:
- Cache hit within TTL -> returns cached value (no DB round-trip).
- Cache miss or expired -> single `.from('platform_config').select('value').eq('key', key).single()` via `createAdminClient()`.
- On ANY error (network, not-found, parse, type mismatch): returns `fallback`. NEVER throws.
- `invalidatePlatformConfigCache(key)` removes that key's cache entry.
- `invalidatePlatformConfigCache()` (no arg) clears the entire cache Map.
