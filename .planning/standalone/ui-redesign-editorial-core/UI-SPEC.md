**This section is part of the visual contract: every ported component carries the scope class named here.**

**A big-bang in-place replace of the `.theme-editorial` block (D-01) would mutate the tokens Conversaciones-v2-live renders against â†’ instant production regression in Somnio.** This violates Regla 6.

### 2.2 The contract: new system lives under a DISTINCT scope class

- The mock CSS selectors `.theme-editorial X` are ported as `.theme-editorial-v3 X` verbatim (1:1 selector rename, no other change).
- **Exact isolation mechanics (whether the v3 scope also retires the two legacy flags at flip time, and how `(dashboard)/layout.tsx` chooses between `.theme-editorial` and `.theme-editorial-v3`) are research/plan discretion (D-05).** The contract only fixes that the new tokens live under `theme-editorial-v3` and that the legacy block is not edited.
